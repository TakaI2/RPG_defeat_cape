## タスク完了マーク

あなたはプロジェクトマネージャーとして、タスク完了を記録し、進捗を更新します。

### 使い方

```
/complete task-001 "実際にかかった時間(hours)"
```

例:
```
/complete task-001 1.5
```

### タスク

1. **`.claude/tasks.json` を読み込む**

2. **指定されたタスクを完了としてマーク**
   - `status` を `completed` に変更
   - `actualHours` を記録
   - `completedDate` を今日の日付に設定
   - `tasks` 配列から `completedTasks` 配列に移動

3. **進捗率を再計算**
   - 全タスクに対する完了タスクの割合
   - フェーズごとの進捗率

4. **`.claude/tasks.json` を更新**

5. **`PROJECT_STATUS.md` を自動更新**
   - `/progress` コマンドを実行して最新状態を反映

6. **完了通知を表示**
   ```
   ✅ タスク完了: [タスク名]
   ⏱️ 実績時間: [X]時間
   📊 全体進捗: [X]% → [Y]%
   🎯 残りタスク: [N]個
   ```

### 注意事項

- タスクIDが存在しない場合はエラーを表示
- 既に完了済みのタスクの場合は警告を表示
- 依存関係があるタスクの場合、依存先が完了しているか確認

**IMPORTANT**:
- 必ず tasks.json を読み書きしてください
- PROJECT_STATUS.md も自動更新してください
- 日本語で出力してください
